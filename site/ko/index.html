Eclipse Collections의 개념

Eclipse Collections는 Java 개발자에게 더 나은 컬렉션 프레임워크를 제공합니다.

이 프레임워크는 풍부한 API, 원시 타입 컬렉션 및 메모리 효율적인 컨테이너를 제공합니다.

Eclipse Collections 학습하기

Eclipse Collections Kata는 Eclipse Collections API를 배우는 재미있는 방법입니다.

Kata는 Eclipse Collections의 다양한 기능을 소개하는 일련의 단계별 연습입니다.

Eclipse Collections 코드베이스에 기여하는 방법을 배우고 싶다면 Eclipse Collections 개발자 가이드를 확인하세요.

  • Eclipse Collections Kata
  • Eclipse Collections 개발자 가이드

Eclipse Collections의 역사

Eclipse Collections는 Goldman Sachs에서 개발한 Caramel(GS Collections)의 오픈 소스 버전으로 시작되었습니다.

2015년 Eclipse Foundation에 기부되었고, 이제 Eclipse Collections로 알려져 있습니다.

참조 가이드

Eclipse Collections 참조 가이드는 Eclipse Collections 라이브러리의 주요 기능을 설명합니다.

이 가이드는 Eclipse Collections의 다양한 컬렉션 타입, 풍부한 API 및 유틸리티 클래스에 대한 개요를 제공합니다.
